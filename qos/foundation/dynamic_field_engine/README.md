# Dynamic Field Engine

Canonical home for promoted kappa, signature, regime, mode, topology, and
prediction primitives.

Lineage is preserved by version families:

- `v07_regimes`: regime detection and persistent phase transitions.
- `v08_modes`: mode extraction, tracking, lifecycle, interactions.
- `v09_topology`: topology graph experiments.
- `v10_prediction`: predictive warning and confidence signals.
- `v11_decision_signals`: foundation-level decision signals, not QCS control.

The stable import surface is `qos.foundation.dynamic_field_engine.core`.
It contains pure tensor transforms only. Plotting, generated artifacts,
benchmark runners, diagnostics, and control policies remain outside this
package.
